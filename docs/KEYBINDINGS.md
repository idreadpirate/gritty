# Keybindings

## Panes

| Action | Keys |
|---|---|
| Split right (left/right panes) | `Ctrl+Shift+D` |
| Split down (top/bottom panes) | `Ctrl+Shift+E` |
| Move focus | `Ctrl+Shift+←↑↓→` |
| Resize focused pane | `Ctrl+Alt+←↑↓→` (Right/Down grow, Left/Up shrink) |
| Rename focused pane | `Ctrl+Shift+R` |
| Close focused pane | `Ctrl+Shift+W` (or type `exit`) |

## Tabs

| Action | Keys |
|---|---|
| New tab | `Ctrl+Shift+T` |
| Next tab | `Ctrl+Tab` |
| Jump to tab N | `Ctrl+1` … `Ctrl+9` |
| Switch tab (mouse) | Click the tab in the bar |

## Clipboard & scrollback

| Action | Keys |
|---|---|
| Copy | `Ctrl+Shift+C` (or drag-select — auto-copies) |
| Paste | `Ctrl+Shift+V`, or right-click |
| Scroll | Mouse wheel (typing returns to the bottom) |

## Command palette

`Ctrl+Shift+P` opens a fuzzy-searchable list. Type to filter, `↑/↓` to move,
`Enter` to run, `Esc` to close. Commands:

- split right / split down
- close pane / rename pane
- new tab / next tab / previous tab
- toggle broadcast input
- toggle seamless mode
- save session / load session

## Mouse

| Action | Gesture |
|---|---|
| Resize panes | Drag the border between panes |
| Switch tab | Click a tab |
| Select + copy | Drag in a pane |
| Paste | Right-click |
| Focus a pane | Click in it |
